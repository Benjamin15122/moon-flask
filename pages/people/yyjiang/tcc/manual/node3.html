Prerequisites

To compete in the Topcoder Algorithm contests, Java Runtime and Topcoder Contest Arena are required.

This cheat program runs both in Linux and in Windows environment as a plugin for Oracle's VisualVM. For Linux users, make sure /usr/bin/timeout exists1. For Windows users, Cygwin is required and Cygwin commands must be available via PATH environment variable. For both users, g++, GNU make and timeout are required to run the system test.
