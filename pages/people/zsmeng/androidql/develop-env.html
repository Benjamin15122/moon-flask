搭建 AndroidQL 开发环境¶

下载并编译 AndroidQL 源代码¶

请参考从源码安装 AndroidQL。

在 Eclipse 中导入项目¶

推荐使用 Eclipse CDT 集成开发环境。AndroidQL 项目为 C++ 代码，可以直接使用 Eclipse CDT 导入。但是由于 AndroidQL 是 Android Runtime 的一部分，直接导入 AndroidQL 代码后，Eclipse 会报告大量的语法错误，且绝大多数代码无法进行跳转。为了读写代码的方便，可以将整个 Android Runtime 项目导入 Eclipse 中。

在编译过一次 AndroidQL 项目之后，AndroidQL 的代码会被复制到 AOSP 项目中。使用 Eclipse 导入 <AOSP_HOME>/art 目录。可以看到 <AOSP_HOME>/art/runtime/aql 目录下包含了 AndroidQL 项目的所有源代码。这里 <AOSP_HOME>/art 目录就是 Android Runtime 项目的所有代码。等待 Eclipse 自动建立 index 之后，就可以正常使用代码跳转等功能。此时 Eclipse 仍会报很少量的语法错误，这是正常现象。
